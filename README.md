# AirScan-QR 📡

**AirScan-QR** 是一款专为**封闭式网络**、**物理隔离环境**（Air-gapped computers）设计的高效文件传输工具。它通过动态二维码序列流，打破网络限制，实现数据在屏幕与摄像头之间的无线传输。

## 🌟 核心痛点解决

在内网、实验室、或生产环境等物理隔离场景下，传统的 U 盘拷贝面临安全风险或管控限制。**AirScan-QR** 提供了一种无需网络、无需驱动、不留物理痕迹的传输方式：
- **穿透隔离**：只要有屏幕和摄像头，就能传输数据。
- **无感适配**：手机扫码即传，无需安装任何客户端应用。
- **全格式支持**：自动处理 Base64 编码，支持图片、文档、压缩包等所有文件类型。

## ✨ 功能亮点

- ⚡ **高速流传输**：基于 `qrious` 优化算法，支持 1-30 FPS 动态帧率调节，大幅提升传输带宽。
- 🧩 **动态分片技术**：单帧容量支持 200-800 Byte 自由调节，根据摄像头性能寻找最佳平衡点。
- 🔄 **智能补帧/重传**：可视化缺失帧清单，点击特定帧序号即可快速定位重传，解决扫码跳帧问题。
- 📄 **完整文件名还原**：支持超长文件名及后缀名的无损传输，确保文件下载后开箱即用。
- 🎨 **极简交互**：基于 Tailwind CSS 构建的响应式界面，发送/接收一键切换。

## 🚀 传输演示

1. **发送端**（内网 PC）：
   - 选择文件 -> 调整单帧容量（建议 350） -> 开始广播。
2. **接收端**（手机/另一台 PC）：
   - 扫码开始 -> 观察进度矩阵 -> 补齐缺失帧。
3. **完成**：
   - 自动合并分片，一键保存完整文件。

## 🛠️ 技术原理

项目基于“二维码流水线协议”：
`TaskID | FullFileName | TotalFrames | CurrentIndex | DataPayload`

- **离线运行**：所有逻辑均在浏览器本地执行，不向任何服务器发送数据。
- **数据一致性**：通过索引校验确保分片合并顺序 100% 准确。

## 📦 部署建议

由于该项目为单 HTML 文件架构，你可以：
1. **直接访问**：通过 GitHub Pages 部署的链接直接访问。
2. **本地保存**：将本项目的 `index.html` 直接另存到本地或内网静态服务器。
3. **完全离线**：通过 U 盘一次性将该 HTML 放入隔离环境，即可永久离线使用。
